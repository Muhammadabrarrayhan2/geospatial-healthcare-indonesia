# Geospatial Healthcare Analysis in Indonesia
Geospatial analysis of healthcare accessibility in Indonesia using machine learning (K-Means) and visualization tools.

# Overview
This project analyzes disparities in healthcare accessibility across Indonesian provinces using geospatial data science techniques. The goal is to identify regions with limited healthcare access and provide insights for improving healthcare infrastructure distribution.

# Objectives
- Analyze the distribution of healthcare services across Indonesia
- Identify disparities between regions (Java vs non-Java regions)
- Detect underserved areas using clustering techniques
- Provide data-driven insights for healthcare policy and planning

# Methods
- Data preprocessing and cleaning
- Feature scaling (MinMaxScaler)
- Clustering using K-Means
- Geospatial visualization using GeoPandas and Folium

# Results & Insights
- Healthcare services are heavily concentrated in Java (e.g., DKI Jakarta, West Java, East Java)
- Eastern regions of Indonesia show significantly lower healthcare accessibility
- Clustering results reveal distinct groups of provinces with high and low healthcare access
- The distribution is influenced by economic development and population density

# Impact
This analysis highlights the need for:
- More equitable healthcare infrastructure distribution
- Policy interventions in underserved regions
- Expansion of digital healthcare solutions such as telemedicine


# Tools Used
- Python
- Pandas, NumPy
- GeoPandas, Folium
- Scikit-learn (K-Means)
- Matplotlib / Seaborn
